웹의 Back/Front 를 넘나들며 경험을 쌓았습니다.

프론트단의 **연동업무**와 **CMS 시스템 개발**, **백엔드 로직, API 개발**을 담당했습니다.

목적에따라 **새로운 언어나 프레임워크**에 대한 학습에 대해 항상 열린 생각을 가지고 있습니다.
